# ReduxDemo
Demo of React state and Redux state for identical todo lists

## How to use (command line):
1. ```console
   git clone https://github.com/jacoblurie29/ReduxDemo
    ```
2. ```console
   cd client
    ```
4. ```console
   npm install
    ```
4. ```console
   npm start
    ```
##### View in your brower at localhost:3000
